# Todo List App

A simple and responsive Todo List application built using **HTML**, **CSS**, **JavaScript**, and **React**.

## Features

* Add new tasks
* Task up, down
* Delete tasks
* Responsive user interface
* Fast and simple task management

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* React.js

## Project Structure

```bash
src/
 ├── components/
 ├── App.jsx
 ├── ToDoList.jsx
 ├── index.css
```
